# CSS練習 — ビジュアル問題集

初心者向けのCSS練習問題集です。模範解答の見た目を再現するCSSを書いて学習します。

## 使い方

1. `css_practices.html` をブラウザで開く
2. 各問題の「HTMLをコピー」ボタンをクリック
3. [CodePen](https://codepen.io/pen/) などのHTML欄に貼り付け
4. CSS欄に自分でCSSを書いて、見本と同じ見た目を再現する
5. 答え合わせしたいときは「解答CSSを見る」ボタンをクリック

## 問題カテゴリ

| カテゴリ | 問題数 | 内容 |
|---------|--------|------|
| テキスト装飾 | 3問 | 見出し・段落、リンク、リスト |
| UIパーツ | 3問 | ボタン、バッジ、入力フィールド |
| 余白/ボックス | 2問 | カード、余白のリズム |
| レイアウト | 3問 | Flexbox、Grid、2カラム |
| 位置 | 2問 | sticky、fixed |
| レスポンシブ | 2問 | メディアクエリ |
| トランジション | 2問 | hover時のアニメーション |

## 学習のポイント

- まずは見本をよく観察する
- 解答を見る前に自分で試す
- 完全に同じでなくても、近い見た目になればOK
- 分からないプロパティは調べながら進める
